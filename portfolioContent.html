Augmented Reality for
Interior Design

An augmented reality mobile application for interior design

This project was developed as my final year project. The objective of the application is to help solve the challenges faced when purchasing furniture and designing a room/space. This application allows the user to select and place furniture and decoration in an environment through AR, there are also additional features such as changing the color of the wall and floor and also creating a collaborative session to design a room with multiple users.

This project was fully developed in Unity with AR Foundation and ARKit.
